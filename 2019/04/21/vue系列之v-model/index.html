vue系列之v-model

发表于 2019-04-21

前言

v-model是Vue用于表单元素上创建双向数据绑定、本质是一个语法糖，在单项数据绑定的基础上，增加了监听用户输入事件并更新数据的功能。

作用

限制 在‘input’、‘select’、‘textarea’、compinents 中使用，

  • .lazy 取代 input 监听change 事件，
  • .number 输入字符串转为有效的数字，
  • .trim 输入首尾空格过滤

正常情况下使用，可以实现双向绑定的效果，从效果上来看，这背后做了两个动作：

  1. v-bind绑定一个value属性；
  2. v-on绑定一个input事件；
