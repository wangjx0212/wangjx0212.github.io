vue_hash模式下，安卓刷新失效

发表于 2019-11-22

前言

vue hash 模式下，安卓环境下刷新 无效问题

内嵌App H5 ，需求是在发布完内容后，通知 app端完成刷新页面；
定义一个全局函数，以供App端调用，

location.reload();
ios 环境下没问，
Android环境下怎没起作用

网上查询后，说是 hash模式下，浏览器只认得 # 号前面的内容，但是使用一下方法：
window.location.href = location.href+’?time=’+((new Date()).getTime());
还是没效果，因为这样子添加的还是在 # 号 后面的内容

所以使用了一下的方法

另外的问题，app分享出去的链接，会被自动添加了几个 search 参数，也就是说参数 是加在 # 号前的，会改变原来的链接，导致链接无效
所以需要进行判断且去掉，同样使用于上面添加 search 参数刷新后正常打开衔接
