CSS清除浮动的几种方法

发表于 2017-04-19

前言

在WEB页面中，当容器的高度为auto时，且容器的内容中有浮动（float为left或right）的元素，容器的高度不能自动撑高以适应内容的高度，
使得内容溢出容器外面影响布局，为了防止这个现象出现，而进行的CSS处理。

例如代码

容器没有包围浮动元素，浮动元素不占空间

清除浮动方法：

方法一 ：使用带clear属性的空元素

在浮动元素后使用一个空元素，并在css中赋予 .clear{ clear:both; }属性即可清理浮动。

优点 ：简单，代码量少，浏览器兼容性好
缺点 : 需要添加大量无语义的html元素，代码不够优雅。

方法二 ：使用CSS的overflow属性
给浮动元素的容器添加overflow:hidden或者overflow:auto，可以清除浮动，但在IE6中需要触发hasLayout

方法三 ：给浮动的元素容器添加浮动

不推荐，这样会使其整体浮动，影响布局

方法四 ：使用邻接元素处理
什么都不做，给浮动元素后面的元素添加clear 属性，类似方法一

方法五 ：使用CSS的伪元素:after

通过css伪元素在容器的最后添加一个看不见的空格，并且赋予clear属性来清除浮动，为了兼容低版本IE，给这这个clearfix添加zoom:1；
触发haslayout,推荐使用此方法

通过以上例子，可发现清除浮动的方法可分为两类：

  1. 利用clear属性，
  2. 触发浮动父元素的BFC（Block Formatting Contexts， 块级格式化上下文），使到该父元素可以包含浮动元素。
