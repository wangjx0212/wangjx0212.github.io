使用Vue.js 2.0搭建单页应用-环境安装

发表于 2017-04-01

前言

如题，主要是记录下vue2的环境安装和配置。

1.安装node.js，node.js的安装包和源码下载地址为：https://nodejs.org/en/download/ ；
2.安装npm，新版的node里已经集成了npm，可通过 “npm -v”来测试是否成功安装，关于网速的问题，可以安装使用淘宝定制的镜像cnpm代替；
3.安装vue-cil,其实vue的脚手架工具，命令如下：

npm install -g vue-cli

参数-g为全局安装，所以这个命令只需要运行一次就可以了。

接着使用webpack去项目模块打包，预处理，热加载，编译vue文件；关于webpack的碉堡特性，后续会有一篇专门的文章进行记录。

vue init webpack vue-myproject


cd vue-myproject

npm install

接着使用，npm run dev在热加载中运行,浏览器会自动打开一个窗口，就可以看到实际的效果了。
