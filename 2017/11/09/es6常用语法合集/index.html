es6常用语法合集

发表于 2017-11-09

前言

ES6是2015年6月正式发布的JavaScript语法标准，低端浏览器需要使用babel语法转换器。热门流行的库基本都基于ES6构建。

新的变量声明方式let 和 const

和var的不同的是新的变量声明方式带来了块作用域和不再具备变量提升

块作用域：

变量提升：

不允许重复声明：

let声明一个值会被改变的变量，const声明一个值不会被改变的变量,称之为常数变量, 所以const一旦声明变量就必须立即初始化，只声明不赋值，会报错

const 保证的并不是变量的值不能改动，而是指向变量的那个内存地址不得改动

es6 声明变量有6种：
var(es5)、function(es5)、 let、const、import、class

模板字符串

使用反引号``将整个字符串包裹起来，当中使用${}来包裹直接写变量或者写表达式。
多行字符串，告别+拼接字符串。

函数扩展

参数默认值

函数参数变量默认声明，所以不能再用let和const进行声明，所以使用参数默认值时，函数不能有同名参数。
参数默认值不是传值，而是每次都重新计算默认值表达式的值，其实它就是惰性求值。

箭头函数

箭头函数可以替换函数表达式、但不能替换函数声明

  • 箭头函数体内的this对象是定义时所在额对象，不是使用时所在的对象；
  • 不可以当做构造函数，不能使用new命令；
  • 不存在arguments对象，可以用rest 参数代替

结构赋值要求等号右边必须是个对象，如果等号是undefined或者null，会报错，因为他们无法转为对象。
还有结构赋值必须是最后一个参数，否则也会报错。

Extends继承

在继承的构造函数中，必须调用一次super方法，表示构造函数的继承，和ES5中使用call&apple继承构造函数是一样的功能。

super还可以直接调用父级的原型方法。比如：super.getName
