Redux是什么

Redux专注状态管理，和react view层解耦， 单一状态、单向数据流
核心的概念：store、state、action、reducer、subscribe
主要功能：
store 管理着所有的状态记录state，如需要改变的时候需要定义action使用dispatch提交至reducer，reducer拿到state和action后，生成新的state;
使用方法：
1.通过reducer新建store，通过store.getState获取状态；
2.需要更改状态时，通过store.dispatch(action)来修改状态；
3.reducer函数接受state和action，返回新的state，可以使用store.subscribe监听每次的修改重新渲染。

Redux和react的一起使用。

使用方法 ：
1.store.dispatch方法传递传递给组件内部调用修改状态；
2.store.subscribe订阅render()函数，每次修改重新渲染；
3.Redux相关的内容，在单独的index.redux.js进行管理处理。
