CSS实现元素垂直居中的几个常用方法

发表于 2017-03-29

前言

前端开发的时候，盒子模型是常常用到的，其中，居中会分为水平居中和垂直居中；水平居中比较容易实现，一般直接设置元素的margin: 0 auto, 就可以实现了，但垂直居中就比较复杂一些，以下记录了几种常用的水平垂直居中的方法

Box垂直水平居中

html Demo :

1.使用margin固定高宽居中

这种方法完全是靠宽高和margin计算拼出来的，使用不是很灵活。

2.父元素relative，子元素absolute ，使用负margin实现居中

需要知道具体的宽高，限制比较大

3.父元素relative，子元素absolute ，使用transform实现居中

这个和用负margin的方法比较，就是可以不需要知道具体的宽高，IE9下不兼容。

4.利用绝对定位居中

非常实用的一种方法

5.使用table-cell的方式居中

6.使用flex的方式居中

7.不确定宽高居中(绝对定位百分数)

在不知道宽高的情况下，使用起来较为灵活，只需保证top和bottom的百分数一样就可以实现垂直居中，left和rigth的百分数一样则就可以实现水平居中。

8.buttom居中方式

这一个比较有趣，但实际情况会比较少使用了，使用buttom作容器，里面的块元素会自动垂直居中，只需要控制下水平居中就可以了。

文字垂直居中

前面了解的是元素垂直水平居中的方法，接下来再了解下，如何使文字垂直居中，还是一样，水平居中比较简单，只要使用text-align: center就可以了;
vertical-align只会作用于同一行内的元素，它的垂直不是相对整个Box而言的，
所以文字垂直居中分为单行和多行来讨论。
